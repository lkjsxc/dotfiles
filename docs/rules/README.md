# Project Rules

Strict rules for managing this dotfiles repository.

## Development
- **Agents**: Only AI agents should modify this repo core structure.
- **Style**: Function over form. No need for human-friendly comments if they bloat the file.

## Structure
- **Recursive**: Use deep directory structures.
- **Docs**: Every directory MUST have a `README.md`.
- **Limits**:
  - Code: < 200 lines.
  - Docs: < 300 lines.

## Maintenance
- Delete unused files immediately.
- Refactor frequently to maintain small file sizes.
