# Arch Linux Dotfiles - Quick Reference

This repository contains comprehensive Arch Linux setup scripts and configurations.

## Quick Setup Commands

```bash
# Full system setup
./setup-scripts/install-system.sh
./setup-scripts/setup-user.sh
./setup-scripts/setup-development.sh

# Deploy configurations
./automation/deploy-configs.sh

# Validate installation
./tests/validate-system.sh
```

## Essential Commands Reference

### System Management
```bash
# Update system
sudo pacman -Syu

# Install packages
sudo pacman -S package-name
yay -S aur-package-name

# Remove packages
sudo pacman -Rns package-name
```

### Development Setup
```bash
# Git configuration
git config --global user.email "lkjsxc@gmail.com"
git config --global user.name "lkjsxc"

# Python environment
pip install virtualenv black flake8 pytest

# Node.js environment
npm install -g typescript ts-node yarn
```

### Key Installations
- **Editor**: neovim, visual-studio-code-bin
- **Browser**: microsoft-edge-stable-bin
- **Fonts**: noto-fonts-cjk, noto-fonts-emoji, fcitx5-mozc
- **Gaming**: lutris
- **Network**: cifs-utils
- **AI Tools**: opencode

### Directory Structure
- `setup-scripts/` - Installation scripts
- `config-files/` - Configuration templates
- `docs/` - Documentation
- `tests/` - Validation scripts
- `automation/` - Deployment utilities

### Configuration Files
- `~/.bashrc` - Shell configuration
- `~/.gitconfig` - Git settings
- `~/.config/Code/User/settings.json` - VSCode settings

For detailed documentation, see `docs/` directory.
