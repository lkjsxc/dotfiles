# .dotfiles — NixOS configuration repository

This repository contains my NixOS configuration, managed as a flake.

Quick bootstrap

- Update flakes: `nix flake update`
- Rebuild system (test/apply): `sudo nixos-rebuild switch --flake .#main`

Layout

- `configuration.nix` — main system configuration (see `docs/CONFIGURATION.md`).
- `hardware-configuration.nix` — autogenerated hardware profile (see `docs/HARDWARE.md`).
- `flake.nix` / `flake.lock` — flake entry and lock (see `docs/FLAKES.md`).
- `docs/` — additional documentation and contributor notes.

Support

For guidance on editing configuration, flake workflow, and regeneration steps, see the `docs/` directory.
