# Dotfiles and NixOS configuration

This repository contains a NixOS configuration managed as a flake.

Table of contents

- **Configuration**: `configuration.nix` — main system configuration
- **Flake**: `flake.nix` — flake entry for building NixOS (see `docs/FLAKES.md`)
- **Hardware**: `hardware-configuration.nix` — generated hardware profile
- **Docs**: `docs/README.md` — documentation and guides

Flake highlights

- Uses `nixos-unstable` from `nixpkgs`.
- Adds `antigravity-nix` as an input and exposes its default package in system packages.

Quick bootstrap

```bash
nix flake update
sudo nixos-rebuild switch --flake .#main
```

Repository layout

- `configuration.nix` — main system configuration
- `hardware-configuration.nix` — autogenerated hardware profile
- `flake.nix` / `flake.lock` — flake entry and lock
- `docs/` — documentation and guides

See `docs/README.md` for more details and per-subdirectory TOCs.
